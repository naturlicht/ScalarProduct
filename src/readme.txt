Файл ScalarProduct.java реализует скалярное произведение векторов с поддержкой многопоточности. В файле Main.java класс
ScalarProduct используется для реализации расчета произведения матрицы на вектор. Результат есть вектор i-ый компонент
которого есть сумма произведений элемента i-ой строки матрицы на элемент с тем же номером вектора. Расчет каждого
элемента результирующего вектора ведется в отдельном потоке.
